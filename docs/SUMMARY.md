## Индекс примеров

```
examples/
├── 01_basics/
│   ├── hello.ot          — Hello World
│   ├── variables.ot      — Переменные, типы, константы, кортежи
│   ├── operators.ot      — Все операторы с примерами (включая ~, ^)
│   └── input.ot          — Чтение данных из stdin
├── 02_control_flow/
│   ├── conditionals.ot   — if / else / unless
│   ├── loops.ot          — Все виды циклов, for x in arr
│   ├── match.ot          — Сопоставление с образцом и enum
│   └── assert.ot         — assert и assert_eq
├── 03_functions/
│   ├── basics.ot         — Объявление и вызов функций
│   ├── defaults.ot       — Параметры по умолчанию
│   ├── lambdas.ot        — Лямбда-выражения
│   ├── static.ot         — Статические методы и ::
│   ├── recursion.ot      — Рекурсивные алгоритмы
│   └── pipe.ot           — Оператор конвейера |>
├── 04_collections/
│   ├── arrays.ot         — Операции с массивами
│   ├── arrays_iter.ot    — Прямой перебор массивов (for x in arr)
│   ├── tuples.ot         — Кортежи и деструктурирование
│   ├── enums.ot          — Объявление и использование enum
│   └── sorting.ot        — Сортировка пузырьком
├── 05_oop/
│   ├── structs.ot        — struct + impl (стиль Go/Rust)
│   ├── classes.ot        — class + init (стиль Java/C#)
│   ├── traits.ot         — trait и impl Trait for Type
│   ├── operators.ot      — impl Add for Type
│   └── inheritance.ot    — class extends (наследование)
├── 06_stdlib/
│   ├── math_demo.ot      — Демонстрация std/math
│   ├── bits_demo.ot      — Демонстрация std/bits
│   └── algo_demo.ot      — Демонстрация std/algo
├── 07_advanced/
│   ├── features.ot       — Все специальные возможности
│   ├── new_features.ot   — Новый синтаксис v2: трейты, лямбды, кортежи
│   ├── syscall_demo.ot   — Прямые системные вызовы и указатели
│   └── net_demo.ot       — TCP-сетевое программирование
├── 08_projects/
│   ├── calculator/        — Многофайловый проект: калькулятор
│   └── geometry/          — Многофайловый проект: геометрия
├── new_syntax.ot         — Полная демонстрация нового синтаксиса v2
└── optionals.ot          — Операторы ?: и ?. (Elvis и optional chaining)
```
