# Глава 3 — Основы языка

В этой главе рассматриваются фундаментальные строительные блоки любой программы на Orbitron:
переменные, типы, константы, операторы и строки.

---

## 3.1 — Переменные

Переменные объявляются с ключевым словом `var`:

```orbitron
var x = 42;
var pi = 3.14;
var flag = true;
```

Orbitron определяет тип переменной по начальному значению:
- Целочисленные литералы → `int` (64-битное знаковое целое, i64)
- Литералы с плавающей точкой → `float` (64-битное двойной точности, f64)
- `true` / `false` → `int` (1 и 0 соответственно)

### Аннотации типов (необязательно)

Тип можно указать явно. Это чисто документационная функция — компилятор не
проверяет совместимость типов по-другому:

```orbitron
var count:  int   = 0;
var ratio:  float = 1.5;
var speed:  int   = 300;
```

### Переприсвоение

После объявления через `var` переменную можно свободно переприсваивать (без `var`):

```orbitron
var score = 0;
score = 100;
score = score + 50;
```

### Правила именования переменных

- Должно начинаться с буквы или подчёркивания (`_`)
- Может содержать буквы, цифры и подчёркивания
- Чувствительно к регистру: `count` и `Count` — разные переменные
- Только ASCII (кириллица в именах переменных не поддерживается)
- Не может совпадать с ключевым словом

```orbitron
var total_score = 0;    // ОК
var _temp       = 99;   // ОК — ведущее подчёркивание допустимо
var x2          = 10;   // ОК
// var 2x = 10;         // ОШИБКА — начинается с цифры
```

## 3.2 — Константы

Константы объявляются с ключевым словом `const`. Значение должно быть числовым литералом:

```orbitron
const MAX_SIZE:   int   = 1000;
const TAX_RATE:   float = 0.2;
const SCREEN_W:   int   = 1920;
const SCREEN_H:   int   = 1080;
```

### Свойства констант

- **Неизменяемы** — нельзя переприсвоить после объявления
- **Любая область видимости** — можно объявлять на верхнем уровне или внутри функции
- **Видны везде** — константы верхнего уровня доступны всем функциям в том же файле и в файлах, которые его импортируют
- **Значение должно быть литералом** — не может быть вычисляемым выражением вроде `2 * 512`

```orbitron
const MAX: int = 100;

func main() {
    const LOCAL: int = 50;   // константа уровня функции
    println(MAX);             // 100
    println(LOCAL);           // 50
}
```

> **Примечание:** Для настоящих значений π и e в виде `float` используйте
> `import "std/math"` — там они вычислены на этапе выполнения.

## 3.3 — Типы данных

В Orbitron два примитивных числовых типа:

| Тип | Размер | Диапазон | Примеры |
|-----|--------|----------|---------|
| `int` | 64-бит знаковый | −9.2 × 10¹⁸ до +9.2 × 10¹⁸ | `0`, `42`, `-100` |
| `float` | 64-бит двойной точности | ≈ ±1.8 × 10³⁰⁸ | `3.14`, `2.718`, `-0.5` |

### Булевы значения

В Orbitron нет отдельного типа `bool`. Булевы значения представляются как `int`:

| Литерал | Значение |
|---------|---------|
| `true`  | `1`     |
| `false` | `0`     |

```orbitron
var flag = true;    // flag == 1
var done = false;   // done == 0

if (flag) {
    println("flag истинно");
}
```

Операторы сравнения тоже возвращают `int`:

```orbitron
var result = 5 > 3;   // result == 1 (не булево, просто int 1)
```

### Повышение типа

Когда в одном арифметическом выражении встречаются `int` и `float`,
`int` автоматически повышается до `float`:

```orbitron
var a: int   = 10;
var b: float = 3.0;
var c = a + b;   // c — float (13.0), a был повышен
```

## 3.4 — Арифметические операторы

| Оператор | Значение | Пример | Результат |
|----------|----------|--------|-----------|
| `+` | Сложение | `3 + 4` | `7` |
| `-` | Вычитание | `10 - 3` | `7` |
| `*` | Умножение | `6 * 7` | `42` |
| `/` | Деление | `10 / 3` | `3` (целочисленное) |
| `%` | Остаток | `10 % 3` | `1` |
| `**` | Возведение в степень | `2 ** 10` | `1024` |
| `-` (унарный) | Отрицание | `-x` | отрицает x |

### Поведение деления

Деление двух целых — **целочисленное** (усекает дробную часть):

```orbitron
var a = 10 / 3;    // 3, а не 3.333...
var b = 7 / 2;     // 3
var c = -7 / 2;    // -3 (округление к нулю)
```

Для деления с плавающей точкой используйте вещественные литералы:

```orbitron
var d = 10.0 / 3.0;   // 3.333...
var e = 7 / 2.0;      // 3.5  (int повышается до float)
```

### Оператор возведения в степень `**`

```orbitron
var p1 = 2 ** 10;    // 1024
var p2 = 3 ** 4;     // 81
var p3 = 2.0 ** 0.5; // ≈ 1.414 (квадратный корень как float)
```

Внутри `**` вызывает функцию `pow()` из `libm`. При целочисленных операндах
результат приводится обратно к `int`.

---

## 3.5 — Операторы сравнения

Сравнение возвращает `1` (истина) или `0` (ложь):

| Оператор | Значение | Пример | Результат |
|----------|----------|--------|-----------|
| `==` | Равно | `5 == 5` | `1` |
| `!=` | Не равно | `5 != 3` | `1` |
| `<` | Меньше | `3 < 5` | `1` |
| `<=` | Меньше или равно | `5 <= 5` | `1` |
| `>` | Больше | `7 > 3` | `1` |
| `>=` | Больше или равно | `4 >= 4` | `1` |

```orbitron
var x = 42;
var is_positive = x > 0;      // 1
var is_even     = x % 2 == 0; // 1
var is_big      = x >= 100;   // 0
```

---

## 3.6 — Логические операторы

| Оператор | Значение | Пример |
|----------|----------|--------|
| `&&` | Логическое И | `a > 0 && b > 0` |
| `\|\|` | Логическое ИЛИ | `a == 0 \|\| b == 0` |
| `!` | Логическое НЕ | `!flag` |

Любое ненулевое значение считается истинным:

```orbitron
var a = 5;
var b = 0;

if (a > 0 && b == 0) {
    println("a положительное и b равно нулю");
}

if (a == 0 || b == 0) {
    println("хотя бы одно равно нулю");
}

if (!b) {
    println("b ложное (ноль)");
}
```

## 3.7 — Составное присваивание

| Оператор | Эквивалент |
|----------|-----------|
| `x += n` | `x = x + n` |
| `x -= n` | `x = x - n` |
| `x *= n` | `x = x * n` |
| `x /= n` | `x = x / n` |

```orbitron
var total = 0;
total += 10;   // total == 10
total += 20;   // total == 30
total -= 5;    // total == 25
total *= 2;    // total == 50
total /= 5;    // total == 10
```

---

## 3.8 — Приоритет операторов

От наименьшего к наибольшему:

| Уровень | Операторы | Ассоциативность |
|---------|-----------|-----------------|
| 1 (наименьший) | `\|>` | левая |
| 2 | `? :` (тернарный) | правая |
| 3 | `\|\|` | левая |
| 4 | `&&` | левая |
| 5 | `==` `!=` `<` `<=` `>` `>=` | — |
| 6 | `+` `-` | левая |
| 7 | `*` `/` `%` | левая |
| 8 | `-` `!` (унарные) | правая |
| 9 | `**` | правая |
| 10 (наибольший) | `.поле` `.метод()` `[индекс]` | левая |

При сомнениях используйте скобки:

```orbitron
var a = 2 + 3 * 4;       // 14 (не 20) — * имеет приоритет выше
var b = (2 + 3) * 4;     // 20
var c = 2 ** 3 ** 2;     // 2 ** (3**2) = 2**9 = 512 (право-ассоциативно)
```

## 3.9 — Строковые литералы

Строки в Orbitron предназначены только для вывода — они могут появляться
только внутри `println()`. Типа данных "строка" для хранения в переменной пока нет.

```orbitron
println("Привет, мир!");
println("Строка с \"экранированными\" кавычками");
println("Табуляция:\tздесь");
```

Стандартные экранирующие последовательности: `\"`, `\\`, `\n`, `\t`.

---

## 3.10 — Строковая интерполяция

Синтаксис `$"..."` позволяет встраивать значения переменных прямо в строку.
Это основной способ вывода вычисленных значений:

```orbitron
var x = 42;
var pi: float = 3.14;
const VER: int = 2;

println($"x = {x}");         // x = 42
println($"pi = {pi}");       // pi = 3.140000
println($"версия: {VER}");   // версия: 2
```

### Правила интерполяции

- Интерполировать можно переменные и константы типов `int` и `float`
- Синтаксис: `{имя_переменной}` внутри `$"..."`
- Можно использовать только внутри `println()`
- В одной строке допускается несколько вставок

```orbitron
var a = 10;
var b = 20;
var sum = a + b;
println($"a={a}, b={b}, сумма={sum}");   // a=10, b=20, сумма=30
```

> **Ограничение:** Выражения типа `{a + b}` внутри интерполяции не поддерживаются —
> только имена переменных или констант. Сначала вычислите значение в переменную,
> затем интерполируйте.

## 3.11 — Комментарии

Комментарии игнорируются компилятором:

```orbitron
// Однострочный комментарий

/*
 * Многострочный блочный комментарий.
 * Можно писать сколько угодно строк.
 */

var x = 42;  // встроенный комментарий после кода
```

## Итоговая таблица

| Конструкция | Синтаксис | Примечание |
|-------------|-----------|-----------|
| Переменная | `var name = value;` | Тип выводится |
| Переменная с типом | `var name: int = value;` | Необязательно |
| Константа | `const NAME: int = value;` | Неизменяемая, только литерал |
| Переприсвоение | `name = new_value;` | Без `var` в левой части |
| Составное присваивание | `x += 5;` | Также `-=`, `*=`, `/=` |
| Строка | `println("текст");` | Только для вывода |
| Интерполяция | `println($"x={x}");` | Встраивает int и float |
| Комментарий | `// строка` или `/* блок */` | Игнорируется |

## Пример: всё вместе

```orbitron
// examples/01_basics/variables.ot
// Демонстрация переменных, констант, типов и операторов.

const GRAVITY: int = 10;    // ускорение свободного падения, м/с²
const MASS:    int = 75;    // масса, кг

func main() {
    // Переменные
    var height:   int = 100;   // метры
    var velocity: int = 0;

    // Арифметика
    var force  = MASS * GRAVITY;            // 750 Н
    var energy = MASS * GRAVITY * height;   // 75000 Дж

    // Составное присваивание
    velocity += 30;
    height   -= 10;

    // Оператор степени
    var v_sq = velocity ** 2;   // 900

    // Сравнение
    var is_moving = velocity > 0;   // 1

    // Интерполяция
    println($"Сила: {force} Н");
    println($"Энергия: {energy} Дж");
    println($"Скорость: {velocity} м/с");
    println($"Скорость в квадрате: {v_sq}");
    println($"В движении: {is_moving}");
}
```
